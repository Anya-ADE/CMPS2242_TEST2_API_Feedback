# CMPS2242 TEST 2 API FEEDBACK

For test #2, you are to create an API that is able to accept data from the feedback form as well as send data to the feedback if required. In other words, build the API.

You do not need to concern yourself with building the UI, but if you decide to do so, you can get a maximum bonus of 10 points.

Your API must have endpoints for creating new feedback, showing all feedback, deleting a feedback, updating feedback, listing only names, listing only emails, listing only subject, and listing only message.


1. A database to be created to store the feedback data.

2. Use SQL migrations to create that database.

3. Data to be JSON encoded and decoded. Use the readJSON and writeJSON functions I gave you

4. Make sure the code is in separate files. I do not want to see one main.go file.

5. You have pre-prepared curl commands to test your endpoints.
